Digital Tools for Cloud-based Data Management

Joao Silva

L1// The Village Simulator

This exercise was a gamified attempted to mimic how a grasshopper definition would run on the web, using only jpegs controlled by the sliders

L2// Patio Layout

This exercise was an experiment with loading 3dm files and interacting with them through the browser. In this case, a control panel allows you to toggle the visibility of two layers where the different design options for a school patio live.

L3// AnyTable

This exercise is the first attempt at running a grasshopper definition on the web for a table configurator. Or ALL the tables.

PIN-UP// The Dieckmann Generator

Erich Dieckmann (1986-1944) was one of the most prominent designers from the Bauhaus. His profile chairs studies was the inspiration for a design tool that allows you to appropriate his designs and build your own.
